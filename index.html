üöÄ Quick Start

Get started in less than 5 minutes with this simple example:

from QuantResearch import fetch_data, Rsi, plot_rsi

# Step 1: Fetch stock data
ticker = "AAPL"
data = fetch_data(ticker, start_date="2023-01-01", end_date="2024-01-01")

# Step 2: Calculate RSI
rsi = Rsi(data['Close'], period=14)

# Step 3: Visualize with candlestick chart
plot_rsi(data=data, rsi=rsi, period=14, ticker=ticker, kind='candle')

RSI indicator displayed with candlestick price chart showing overbought/oversold levels

üìä RSI - Relative Strength Index

Rsi(price, period=14)

Momentum oscillator measuring the magnitude of recent price changes.

Interpretation:
  • RSI > 70 ‚Üí Overbought (potential sell signal)
  • RSI < 30 ‚Üí Oversold (potential buy signal)
  • 30-70 ‚Üí Neutral zone
from QuantResearch import fetch_data, Rsi

data = fetch_data("AAPL", "2023-01-01", "2024-01-01")
data['rsi'] = Rsi(data['Close'], period=14)

# Generate signals
oversold = data['rsi'] < 30  # Buy signal
overbought = data['rsi'] > 70  # Sell signal

print(f"Current RSI: {data['rsi'].iloc[-1]:.2f}")
print(f"Buy signals: {oversold.sum()}")
print(f"Sell signals: {overbought.sum()}")

RSI indicator showing buy and sell signals

üìà MACD - Moving Average Convergence Divergence

macd(price, short_period=12, long_period=26, signal_period=9)

Trend-following momentum indicator that shows the relationship between two moving averages.

from QuantResearch import fetch_data, macd

data = fetch_data("AAPL", "2023-01-01", "2024-01-01")
macd_line, signal_line, hist = macd(data['Adj Close'])

# Detect crossovers
bullish = (macd_line > signal_line) & (macd_line.shift(1) <= signal_line.shift(1))
bearish = (macd_line < signal_line) & (macd_line.shift(1) >= signal_line.shift(1))

print(f"Bullish signals: {bullish.sum()}")
print(f"Bearish signals: {bearish.sum()}")

MACD with signal line and histogram showing trend strength

üìâ Bollinger Bands

bb_bands(price, period=20, num_std=2)

Volatility bands placed above and below a moving average. Useful for identifying overbought/oversold conditions.

from QuantResearch import fetch_data, bb_bands

data = fetch_data("AAPL", "2023-01-01", "2024-01-01")
upper, mid, lower = bb_bands(data['Adj Close'], period=20, num_std=2)

data['bb_upper'] = upper
data['bb_mid'] = mid
data['bb_lower'] = lower

# Calculate band width for squeeze detection
band_width = (upper - lower) / mid
squeeze = band_width < band_width.quantile(0.25)
print(f"Squeeze periods detected: {squeeze.sum()}")

Bollinger Bands showing volatility zones around price movement

üìè ATR - Average True Range

atr(data, period=14)

Volatility indicator that measures the average range of price movement.

from QuantResearch import fetch_data, atr

data = fetch_data("AAPL", "2023-01-01", "2024-01-01")
data['atr'] = atr(data, period=14)

print(f"Average ATR: {data['atr'].mean():.2f}")
print(f"Current ATR: {data['atr'].iloc[-1]:.2f}")

# Plot ATR
data['atr'].plot(title=f'ATR for AAPL')

Average True Range showing market volatility over time

üìä SMA - Simple Moving Average

sma(price, period=9)

Average price over a specified period, useful for identifying trend direction.

from QuantResearch import fetch_data, sma

data = fetch_data("AAPL", "2023-01-01", "2024-01-01")
data['sma_20'] = sma(data['Adj Close'], period=20)
data['sma_50'] = sma(data['Adj Close'], period=50)

# Golden Cross detection
golden_cross = (data['sma_20'] > data['sma_50']) & (data['sma_20'].shift(1) <= data['sma_50'].shift(1))
print(f"Golden Cross signals: {golden_cross.sum()}")

Simple Moving Average overlaid on candlestick chart

üìà EMA - Exponential Moving Average

ema(price, period=9)

Weighted average giving more importance to recent prices, more responsive than SMA.

from QuantResearch import fetch_data, ema

data = fetch_data("AAPL", "2023-01-01", "2024-01-01")
data['ema_12'] = ema(data['Adj Close'], period=12)
data['ema_26'] = ema(data['Adj Close'], period=26)

print(f"Current EMA 12: {data['ema_12'].iloc[-1]:.2f}")
print(f"Current EMA 26: {data['ema_26'].iloc[-1]:.2f}")

Exponential Moving Average showing trend with less lag

üìä DEMA & TEMA

demma(price, period=9) & temma(price, period=9)

DEMA: Double Exponential Moving Average - smoother with reduced lag
TEMA: Triple Exponential Moving Average - smoothest with minimal lag

from QuantResearch import fetch_data, demma, temma

data = fetch_data("AAPL", "2023-01-01", "2024-01-01")
data['dema'] = demma(data['Adj Close'], period=20)
data['tema'] = temma(data['Adj Close'], period=20)

print("Advanced moving averages calculated successfully!")

DEMA and TEMA comparison showing reduced lag in trend following

üíπ RVWAP - Rolling Volume-Weighted Average Price

RVWAP(high, low, close, volume, period=20)

Price benchmark that reflects both volume and price data, useful for institutional trading levels.

from QuantResearch import fetch_data, RVWAP

data = fetch_data("AAPL", "2023-01-01", "2024-01-01")
data['rvwap'] = RVWAP(data['High'], data['Low'], data['Close'], 
                      data['Volume'], period=20)

print(f"Current RVWAP: {data['rvwap'].iloc[-1]:.2f}")

Rolling VWAP showing volume-weighted price levels

üìä RSI Visualization

# Line chart
plot_rsi(rsi=rsi, period=14, ticker="AAPL", kind='line')

# Candlestick chart with RSI
plot_rsi(data=data, rsi=rsi, period=14, ticker="AAPL", kind='candle')

üìâ Bollinger Bands Visualization

üìä Moving Averages Visualization

plot_moving_averages(data, price, sma_val, ema_val, dema_val, tema_val, ticker, kind='line')

Compare multiple moving averages on a single chart.

from QuantResearch import fetch_data, sma, ema, demma, temma
from QuantResearch import plot_moving_averages

data = fetch_data("AAPL", "2023-01-01", "2024-01-01")
data['sma'] = sma(data['Adj Close'], period=20)
data['ema'] = ema(data['Adj Close'], period=20)
data['dema'] = demma(data['Adj Close'], period=20)
data['tema'] = temma(data['Adj Close'], period=20)

# Candlestick with all moving averages
plot_moving_averages(data=data, sma_val=data['sma'], 
                     ema_val=data['ema'], dema_val=data['dema'],
                     tema_val=data['tema'], ticker="AAPL", kind='candle')

Multiple moving averages (SMA, EMA, DEMA, TEMA) compared on candlestick chart
